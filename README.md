# Привет! Я Лещева Ольга 👋

### Из экспертного учета в DevOps-инженерию 🏗️☁️

Я посвятила 35 лет бухгалтерии, где точность, регламенты и системный подход были основой моей работы. Сейчас я трансформирую этот колоссальный опыт в мир **DevOps**, объединяя железную дисциплину финансового контроля с гибкостью современных облачных технологий.

---

### 🛠️ Мой технологический стек (после обучения в GeekBrains):
- **Infrastructure as Code:** Terraform, Ansible
- **CI/CD:** GitLab CI, GitHub Actions
- **Containerization:** Docker, Kubernetes (K8s)
- **Monitoring & Logging:** Prometheus, Grafana, ELK Stack
- **OS & Scripting:** Linux (Bash), Python

---

### 💼 Почему мой бэкграунд полезен в DevOps?
*   **FinOps Mindset:** Я как никто другой умею оптимизировать затраты. Облачные ресурсы должны работать эффективно и экономно.
*   **Reliability & Accuracy:** 35 лет без критических ошибок в отчетности научили меня вниманию к деталям, которое необходимо при настройке инфраструктуры.
*   **System Thinking:** Умею видеть сложные взаимосвязи в больших системах и автоматизировать рутину.
